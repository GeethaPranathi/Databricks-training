# Project 2-HR Dashboard using PySpark

## Overview
This project demonstrates basic HR data analysis using **PySpark DataFrame APIs**. It analyzes employee information to generate useful HR insights and reports.

## Technologies Used
- Python
- PySpark
- Apache Spark

## Features
- Count Employees by Gender
- Calculate Average Age Department-Wise
- Analyze Hiring Trends by Year
- Display Employee Distribution City-Wise
- Find Highest Paid Employee(s)

## Dataset
The dataset contains the following columns:
- Employee ID
- Name
- Gender
- Age
- Department
- City
- Salary
- Hire Date

## How to Run
1. Install PySpark:
   ```bash
   pip install pyspark
   ```
2. Run the Python file:
   ```bash
   python hr_dashboard.py
   ```
