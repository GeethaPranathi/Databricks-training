# Project 3-Employee ETL Pipeline using PySpark

## Overview
This project demonstrates a complete **ETL (Extract, Transform, Load) Pipeline** using **PySpark**. It focuses on data cleaning, transformation, optimization, and aggregation to prepare employee data for analytics.

## Technologies Used
- Python
- PySpark
- Apache Spark

## Features
- Remove Duplicate Records
- Handle Missing Data
- Join Multiple Datasets
- Optimize Data Partitions
- Create Aggregated Summary Table
- Complete ETL Pipeline

## Dataset
The dataset contains the following columns:
- Employee ID
- Name
- Department
- Salary
- City

## How to Run
1. Install PySpark:
   ```bash
   pip install pyspark
   ```
2. Run the Python file:
   ```bash
   python employee_etl_pipeline.py
   ```

## Learning Outcomes
- Data Cleaning
- Handling Missing Values
- Removing Duplicates
- DataFrame Joins
- Partition Optimization
- Aggregation and ETL Workflow
