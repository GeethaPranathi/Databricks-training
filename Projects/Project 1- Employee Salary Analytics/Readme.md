# Project 1-Employee Salary Analytics using PySpark

## Overview
This project demonstrates employee salary analysis using **PySpark DataFrame APIs**. It performs various analytical operations on employee data to generate useful business insights.

## Technologies Used
- Python
- PySpark
- Apache Spark

## Features
- Find Top 3 Highest Salaries Department-Wise
- Calculate Average Salary City-Wise
- Categorize Employees into Salary Bands
- Generate Yearly Salary Report
- Find Employees Earning Above Department Average

## Dataset
The dataset contains the following columns:
- Employee ID
- Name
- Designation
- Department
- Salary
- City
- Age
- Joining Date
- Gender

## How to Run
1. Install PySpark:
   ```bash
   pip install pyspark
   ```
2. Run the Python file:
   ```bash
   python employee_salary_analytics.py
   ```
